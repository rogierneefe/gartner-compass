# Werkregels — Token Efficiency

## Model
- Gebruik altijd Sonnet tenzij je vastloopt
- Alleen naar Opus bij complexe redeneerslag die Sonnet niet oplost

## Gedrag
- Schrijf direct code — geen plan, geen samenvatting vooraf
- Geen uitleg tenzij gevraagd
- Beknopte commentaar in code, niet in chat
- Bevestig alleen: bestandsnaam + "klaar" per afgerond bestand

## Context
- Gebruik /compact zodra context >50% vol is
- Gebruik /cost na elke fase om verbruik te checken
- Lees nooit meer bestanden tegelijk dan nodig voor de huidige taak

## Sessie afsluiten
Voeg na elke sessie toe aan ## Sessie-logboek in CLAUDE.md:
- Welke bestanden zijn af ✅
- Welke fase nu open staat
- Max 5 regels
